# Color-Palette-Generator

[![Live Demo](https://img.shields.io/badge/demo-live-blueviolet)](https://red-arachnid.github.io/Color-Palette-Generator/)
The Color Palette Generator is a clean, interactive web application that helps designers and developers instantly create beautiful, mathematically accurate color schemes. By selecting a single base color, users can instantly generate harmonious palettes based on classic color theory algorithms, including:
- Monochromatic
- Analogous
- Complementary
- Split-Complementary
- Triadic

## Technologies Used
- **HTML5:** For the structural layout and semantic markup.
- **CSS3:** Utilizing Flexbox and CSS Grid for a responsive, modern, and aligned user interface.
- **Vanilla JavaScript (ES6+):** For DOM manipulation, event handling, and complex color-math conversions.

## What I Learned
Building this project was a great exercise in solidifying my frontend development skills. Key takeaways include:
- **Color Mathematics:** Understanding how different color models work (HSV, HSL, HEX) and writing the mathematical functions to convert between them.
- **Color Theory:** Translating artistic concepts (like the degrees between complementary colors on a color wheel) into actionable code logic.
- **Event Handling:** Managing UI states, timeouts, and event bubbling (e.g., handling the custom color picker tooltips and copy-success checkmarks).
